O QUE É O SIDA?
É um projeto de tradução simultânea de Libras entre a comunidade surda e o mundo corporativo.
Foi desenvolvido com o propósito de acessibilidade na comunicação interna à empresa, para inclusão dos surdos nas tarefas, reuniões e logísticas dentro do espaço empresarial.
Hoje, o site está hospedado em https://matheusbrugnago.github.io/projeto_sida/.

COMO ELE TRABALHA?

Há dois tipos de mensagens entre a comunidade surda e o mundo, um tenta falar e o outro demonstrar os sinais. Para resolução foi trabalhado em dois canais:

Surdo -> Mundo ( Ativação de IA que traduz os sinais para português escrito, e automaticamente para linguagem falada )
Mundo -> Surdo ( Funcionalidade de Falar para o Surdo e encaminhar português escrito em chat )

1-Faça download dos arquivos para repositório local
2-Clique com botão direito em cima do index.html e rode o Live Server no Chrome
3-Já irá abrir na Aplicação
